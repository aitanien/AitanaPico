Dart: Banking app

Self-commissioned design of an app interface and landing page for a banking application. I used 3D programs to create abstract designs that complemented the interface. The structure of the application is made in two layers, the first in 2D with functional elements representing measurable information. This is overlaid on a second abstract element that adds depth to the interface and represents the user's aspirations emerging from the data.

Autoencargo individual de un diseño de interfaz de app y landing page para una aplicación bancaria. Utilicé programas de 3D para crear diseños abstractos que complementaban la interfaz. La estructura de la aplicación está hecha en dos capas, la primera en 2D con elementos funcionales que representan la información medible. Esta está superpuesta sobre un segundo elemento abstracto que le da profundidad a la interfaz y representa las aspiraciones del usuario que emergen de los datos.
